Etiquetas de Bloque:

• Un elemento en bloque ocupa todo el espacio de su elemento padre "contenedor", creando así un "bloque"

• Los navegadores suelen mostrar el elemento a nivel de bloque con un salto de línea antes y después del elemento.

• Los elementos de bloque sólo deben aparecer dentro del elemento "body".

• Es importante reconocer que tipo de elemento es una etiqueta para poder formatearlo adecuadamente con el CSS

Entre las Etiquetas de Bloque tenemos:

    • article: contenido de articulo
    • aside: contenido adicional
    • audio: reproductor de audio
    • div: división de documento
    • footer: pie de página
    • form: formulario de entrada
    • h1..h6: titulos y subtitulos
    • header: sección o cabecera
    • li: elemento de lista
    • ol: lista ordenada
    • p: párrafo
    • section: sección de una página
    • table: tabla
    • ul: lista no ordenada
